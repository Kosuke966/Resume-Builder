# Resume-Builder
A fully frontend, interactive resume generator that lets users build professional resumes directly in the browser without writing any code. Users can input personal details, skills, experience, and projects, see a live preview, switch between light/dark themes, and download a ready-to-use HTML resume.
 fully frontend **interactive resume generator** that lets users build professional resumes **directly in the browser**. Enter your details, skills, experience, and projects, see a **live preview**, toggle **light/dark theme**, and download a **ready-to-use HTML resume** — no coding required.

---

## Features

-  Live Preview – See changes instantly as you type  
-  Light/Dark Theme – Toggle your preferred theme  
-  Custom JSON Input – Add experience and projects using JSON  
-  Download HTML Resume – Export a complete resume with one click  
-  Frontend Only – Works entirely in the browser
